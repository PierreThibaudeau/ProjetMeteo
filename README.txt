
Ce site web à été réalisé entièrement par moi-même, pendant un BTS qui ne formait pas au web, sur une periode de 4-5 mois.
Veuillez donc excuser les coquilles dans le code si il y en a.
La page la plus interessante du point de vue Php est Données_Archivées

1. Installer le dossier Code_Source sur UWamp ou EasyPhp.
2. Importer les tables du dossier Data_Mysql avec mysql dans une Base "meteo2".
3.1. créez dans mysql un utilisateur MeteoLecture, avec un droit de SELECT
3.2. créez dans mysql un utilisateur MeteoEcriture, avec un droit de DELETE, INSERT et ALTER
4. Se connecter à Internet pour que le module Highchart fonctionne
5. Aller sur "Effacer la Bdd" pour effacer les données et sur "Remplir la bdd" pour en remettre
6. Dans données_Archivées, Entrez les valeurs selon l'interval affiché dans "Remplir la Bdd"
